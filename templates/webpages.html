HTML

Tag Description
html the tag that goes round all the html in the page
head the part of the page that gives information to the browser
title The text to go in the title tab in the browser
link A link to the stylesheet for the page
h1, h2, ... h6 Levels of heading from the main (h1), through the sub heading levels (h2 to h6)
p A paragraph. Leaves space vertically between paragraphs
br Line break. Moves what is after the breake to the next line as part of the same paragraph
a Anchor. A hyperlink to another page using href attribute. Can also be used for bookmarks
ul An unordered list (bulletted)
ol An ordered list (1,2,3 or a,b,c or i, ii, iii etc)
li List item. An item in an ordered or unordered list
table The tag that goes around a table
tr The row of a table
th A heading cell of a table
td A normal cell in a table
img An image to display on the page
